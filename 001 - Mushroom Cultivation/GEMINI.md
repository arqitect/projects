# GEMINI.md

## Directory Overview

This directory contains the project files for a DIY mushroom cultivation project. The goal is to use a budget-friendly and beginner-friendly method using pre-cooked rice bags, commonly known as "Uncle Ben's Tek".

## Key Files

*   **README.md:** This file provides a step-by-step guide for the mushroom cultivation process, including a list of necessary materials and instructions for inoculation, colonization, and fruiting.
*   **001 - project-log.md:** This file tracks the progress of the project, including goals, status, log entries, key decisions, and next steps. It also includes a status of material acquisition.

## Usage

The contents of this directory are intended to be used for tracking and managing the mushroom cultivation project. The `README.md` serves as a guide for the process, and the `001 - project-log.md` is used to document the journey.
